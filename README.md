# Resume Portfolio

A personal portfolio website for Chau Nhu Quynh, built with HTML, CSS, and JavaScript.

## Structure

- `index.html` - main page
- `style.css` - styles
- `script.js` - interactions and scroll animation
- `assets/` - images and media

## How to run locally

1. Open `index.html` directly in your browser.
2. Or use a simple local server, for example:
   ```powershell
   python -m http.server 3000
   ```
3. Visit `http://localhost:3000`.
